finalize(handle, failed) → {Uint8Array|Promise.<?Uint8Array>}

Source:
  • globals.js, line 123
Parameters:
Name Type Description
handle H5Group

Return value of create. This will already have its close method invoked.
failed boolean

Whether an error occurred when writing to the HDF5 file.
Returns:

This should execute clean-up operations when no more write operations are to be performed on the file returned by create.

If failed = true, any existing resources associated with the file may be deleted, and null should be returned, possibly asynchronously.

If failed = false and path was used in create, null should be returned, possibly asynchronously. Otherwise, if path was not used in create, a Uint8Array (or a promise thereof) with the contents of the HDF5 file should be returned, typically for use in {linkcode GlobalFsInterface#write GlobalFsInterface.write}.
