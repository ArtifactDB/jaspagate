(constant) saveObjectRegistry :Array

Description:

  • Registry of saving functions. Each entry should be an array of length 2, containing a Javascript class and its saving function. Each saving function should accept the same arguments as saveObject. Subclasses should be placed after their parents in this array.

(async) readObject(path, metadatanullable, globals, optionsopt)

Description:

  • This function will inspect readObjectRegistry to check if any reader function is supplied for the takane object type at path. If found, it will use that function, otherwise it will fall back to the default functions:

    • readDataFrame, to read DataFrame objects.
    • readSummarizedExperiment, to read SummarizedExperiment objects.
    • readRangedSummarizedExperiment, to read RangedSummarizedExperiment objects.
    • readSingleCellExperiment, to read SingleCellExperiment objects.
